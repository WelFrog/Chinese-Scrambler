隐私政策 (Privacy Policy)

本隐私政策旨在透明地向您说明，当您访问部署在 GitHub Pages 上的“汉字乱序生成器”（welfrog.github.io）时，我们如何处理、收集及保护您的信息。我们非常重视您的隐私，并致力于遵守全球范围内的隐私保护标准。

1. 核心隐私声明：本地处理

我们的核心功能（汉字乱序算法）完全基于客户端 JavaScript 实现。这意味着当您在输入框中输入文本并点击生成时，所有计算处理均在您自己的浏览器内存中完成。我们不会、也无法将您的原始文本上传到任何服务器进行存储或二次处理。 您的内容隐私在技术层面得到了闭环保护。

2. Google AdSense 与 Cookie 使用

为了维持本工具的免费运营与持续开发，本站接入了 Google AdSense 广告服务。Google 作为第三方供应商，会使用 Cookie 来根据用户访问本网站或其他网站的情况投放广告：

  • Google 使用广告 Cookie（如 DoubleClick Cookie）来个性化投放广告，提高广告的相关性。
  • 这些 Cookie 允许 Google 及其合作伙伴根据您在本站及互联网上其他站点的访问轨迹提供精准广告。
  • 您可以通过访问 Google 的 广告设置 页面随时停用个性化广告。

3. 自动收集的非个人数据

虽然我们不收集您的个人身份信息，但我们的托管服务商 GitHub 以及分析工具可能会自动记录一些基础信息，这些信息通常包括：

  • 浏览器类型与版本、操作系统信息。
  • 来源站点（您是从哪个页面跳转而来的）。
  • 访问的时间戳以及在本站停留的时长。

这些数据仅用于统计分析，旨在优化网站的响应速度和用户体验。

4. 第三方链接声明

本站包含指向外部站点（如 GitHub 开源仓库）的链接。我们不对这些第三方站点的隐私实践或内容负责，建议您在访问时查看其对应的隐私声明。

5. 政策更新与联系

我们可能会不时更新本政策以反映技术变化。若您对隐私处理有任何疑问或建议，欢迎通过本项目在 GitHub 上的 Issues 页面提出反馈。

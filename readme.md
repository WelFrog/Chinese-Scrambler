# 汉字乱序生成器

本工具基于心理学效应，通过智能分词算法对中文文本进行“乱序不乱意”的处理，生成具有欺骗性的趣味文本。

## 功能介绍

本工具利用浏览器的 `Intl.Segmenter` 引擎进行中文分词，并将原句进行随机打乱。在保持核心词意基本完整的前提下，对文本进行随机乱序或相邻词汇交叉互换，使人脑在阅读时产生自动修正的效果。

## 使用教程

### 1. 部署与访问

本工具为纯前端实现，无需后端服务器。你可以直接保存并运行根目录下的 `index.html`。

---

### 2. 输入原文

在左侧工作台的 **“输入原文”** 区域粘贴或输入你想要处理的中文段落。

---

### 3. 调整配置参数

在右侧 **“配置中心”** 调整生成逻辑，以达到理想的混淆效果。

参数解释：

* **乱序词数比例**：决定了全文中有多少比例的词汇会被打乱。
* **二词交叉概率**：针对相邻的两个二字词，有一定的概率触发位置互换。如“研究表明”会被替换为“研表究明”。

---

### 4. 自定义白名单

为了防止专有名词（如人名、地名、专业术语）被打乱导致语义彻底丢失，你可以使用白名单功能。

* 在输入框输入需要保护的词汇，点击 **“添加”**。
* 被添加的词汇在生成过程中将保持原样，不受乱序参数影响。

> [!NOTE]
> 
> 程序内置了常见姓氏和虚词的自动保护机制，确保处理后的文本依然符合中文基本的语感。

---

### 5. 生成与导出

点击 **“立即生成”** 按钮，处理结果将实时显示在下方区域。点击 **“复制”** 按钮即可将乱序文本保存至剪贴板。
